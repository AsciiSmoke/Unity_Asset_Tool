
# Unity_Asset_Tool
A tool for extracting, repacking and searching .bundle and .assets files used in the Unity Engine

Created with considerable help from Claude AI. 

**QuickStart:** Double click 'Unity_Asset_Tool.exe', this runs the code in [unity_asset_tool.py](https://github.com/AsciiSmoke/Unity_Asset_Tool/blob/main/unity_asset_tool.py "unity_asset_tool.py").

***Important:** 'Unity_Asset_Tool.exe' and 'unity_asset_tool.py' must be in the same directory.*
